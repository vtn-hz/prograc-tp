# Sistema de Gestión de Clínica

Sistema de gestión diseñado para administrar las operaciones de una clínica privada. Este proyecto implementa funcionalidades básicas para el manejo de pacientes, médicos, turnos e historias clínicas.

## Tecnologías Utilizadas

- **Versión:** Java 1.8
- **Gestión de Dependencias:** Maven
- **IDEs:** IntelliJ y Eclipse
